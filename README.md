# elitalon's dotfiles


## Installing

```bash
git clone https://github.com/elitalon/dotfiles.git && cd dotfiles && ./bootstrap.sh
```

### Upgrading

To update, `cd` into your local `dotfiles` repository and then:

```bash
./bootstrap.sh
```
